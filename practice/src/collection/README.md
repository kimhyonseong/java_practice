# 컬렉션 프레임워크
자료구조를 구현해 놓은 JDK 라이브러리  
Collection + Map  

## Collection
collection - 객체  
여러 원소를 담을 수 있는 자료구조  
  
종류
- List : 객체를 순서에 따라 저장 / 중복 허용
  - ArrayList
  - Vector
  - LinkedList
- Set : 순서에 상관없이 중복 불허 / 순서대로 저장하지 않음 - 정렬가능
  - HashSet
  - TreeSet

## Map
쌍으로 이루어진 객체를 관리  
key-value 구조  
key 중복안됨 
검색을 위한 자료구조
  
종류
- Hashtable
- HashMap
- TreeMap
